# ServingsApp

Приложение для системы заказов с консольным клиентом и библиотеками для взаимодействия с сервером.

## Описание проекта

Проект состоит из нескольких компонентов:

- **Консольное приложение** - клиент для работы с системой заказов
- **Библиотека HTTP-клиента** - для взаимодействия с REST API сервера
- **Библиотека gRPC-клиента** - для взаимодействия с gRPC сервером

## Структура проекта

- `ServingsLib/` - Библиотеки для взаимодействия с сервером (HTTP и gRPC)
- `Servings.ConsoleApp/` - Консольное приложение для управления заказами
- `Servings.Domain/` - Доменная модель
- `Servings.Infrastructure/` - Инфраструктурные компоненты
- `Servings.Application/` - Слой бизнес-логики
- `Servings.MockApi/` - Mock API для тестирования
